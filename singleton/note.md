# 关于单例，多进程，多线程的思考

我觉得如果使用单例模式，那么它能够保证在同一个进程中只有一个对象；如果这个进程有多个线程，那么所有的线程将只能够访问这一个对象；这引发了我关于类，对象，多进程，多线程的思考。
